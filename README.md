# net_control

Десктоп-приложение на PySide6 с нормальной структурой проекта:
- профили
- настройки
- запуск/остановка команды из профиля
- логирование
- сохранение в json
- системный трей

## запуск

```bash
pip install -r requirements.txt
python main.py
```

## пример профиля

Можно указать безопасную команду, например:
- Windows: `ping 1.1.1.1 -t`
- Linux/macOS: `ping 1.1.1.1`
